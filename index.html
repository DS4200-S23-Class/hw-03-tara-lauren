hw-03: Interactive Webpage, and Marks & Channels

  • Visualization 1
    • Marks:
      • Area (state map and electoral vote bar)
      • Points (NE and ME electoral votes)
    • Channels:
      • Colors (represent if majority voted for Clinton (blue) or Trump (red))
      • Shapes (represent shape of each US state)
      • Horizontal position (the amount of electoral votes for each candidate.)
    • Colormaps:
      • Categorical colormap, represents what political party was voted for (blue is democrat, red is republican).



  • Visualization 2
    • Marks:
      • Area (the scoring value bar)
      • Points (they make up the graph)
    • Channels:
      • Position (both, x represents exit velocity, y represents launch angle)
      • Color hue (represents scoring value)
      • Color luminance (to represent LWTS of each data point)
      • Shape (hexagons, represents batted balls)
      • Horizontal position (scoring value bar)
    • Colormaps:
      • Divergent colormap, represents the scoring value of each batted ball. The white in the middle is the average scoring value while the pink represents an extreme to the left (below avg.) and the orange is an extreme to the right (above avg.).

  • Visualization 3
    • Marks:
      • Lines (each bar is a line)
    • Channels:
      • Color hue (each represents a different color of diamond)
      • Position (both, x represents diamond color, y represents total price for that diamond)
    • Colormaps:
      • Categorical colormap, each bar represents a different color of diamond.
